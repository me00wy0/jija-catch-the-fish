# 🐟 Catfish Cove

A charming underwater fishing game where you play as a cat catching fish in the ocean!

## 🎮 How to Play

- **Arrow Keys (← →)** - Move the cat left and right
- **Catch Fish 🐟** - Get 10 points per fish
- **Catch Stars ⭐** - Get 30 points per star (rare!)
- **Reach 300+ points** - Win! The cat is full!
- **Lose all 3 lives** - Game over, the cat swims home

## 🎯 Game Features

✨ Beautiful underwater gradient background  
🐱 Smooth cat movement controls  
🐟 Dynamic fish spawning with sine-wave movement  
🫧 Animated bubbles for atmosphere  
📊 Real-time score and lives tracking  
🎨 Elegant UI with floating title animation  
📱 Responsive fullscreen gameplay
